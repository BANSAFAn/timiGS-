# Installation Guide

TimiGS is available for Windows, Linux, and macOS.

## Windows
1. Download `TimiGS_x64-setup.exe`.
2. Run the installer.
3. Windows SmartScreen may appear (since we are not yet EV signed). Click "More Info" -> "Run Anyway".

## Linux
- **AppImage**: `chmod +x TimiGS.AppImage && ./TimiGS.AppImage`
- **DEB**: `sudo dpkg -i timigs.deb`

## macOS
1. Download `.dmg`.
2. Drag to Applications.
3. Right-click Open if Gatekeeper complains.
